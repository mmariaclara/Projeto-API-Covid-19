Nossa Finalidade e Objeto
Nesse projeto, colocamos em prática os nossos aprendizados sobre JavaScript e API. Nosso objetivo é apresentar de forma simples os dados sobre a pandemia do Covid-19, tanto no Brasil, quanto no mundo. Os dados do mundo foram trazidos atráves do mapa mundi e do gráfico de pizza. Já os dados do Brasil, foram trazidos por uma tabela. Para tal coisa, utilizamos a API, HTML 5, CSS 3 e JavaScript. Os dados utilizados no projeto foram obtidos através da Covid19 Brazil API https://covid19-brazil-api-docs.vercel.app/
Linguagens Utilizadas
  • HTML 5
  • CSS 3
  • JavaScript
  • API
  • Google Charts
